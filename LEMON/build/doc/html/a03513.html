Detailed Description

template<typename M>
class lemon::NegMap< M >

This read-only map returns the negative of the values of the given map (using the unary - operator). Its Key and Value are inherited from M.

If M::Value is int, double etc., then

NegMap<M> neg(m);

is equivalent to

ScaleMap<M> neg(m,-1);

The simplest way of using this map is through the negMap() function.

See also
NegWriteMap

#include <lemon/maps.h>
