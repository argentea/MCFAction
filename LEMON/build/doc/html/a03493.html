Detailed Description

template<typename M1, typename M2>
class lemon::DivMap< M1, M2 >

This read-only map returns the quotient of the values of the two given maps. Its Key and Value types are inherited from M1. The Key and Value of M2 must be convertible to those of M1.

If m1 is of type M1 and m2 is of M2, then for

DivMap<M1,M2> dm(m1,m2);

dm[x] will be equal to m1[x]/m2[x].

The simplest way of using this map is through the divMap() function.

See also
AddMap, SubMap, MulMap
