This function reads a minimum cost flow instance from DIMACS format, i.e. from a DIMACS file having a line starting with

p min

At the beginning, g is cleared by g.clear(). The supply amount of the nodes are written to the supply node map (they are signed values). The lower bounds, capacities and costs of the arcs are written to the lower, capacity and cost arc maps.

If the capacity of an arc is less than the lower bound, it will be set to "infinite" instead. The actual value of "infinite" is contolled by the infty parameter. If it is 0 (the default value), std::numeric_limits<Capacity>::infinity() will be used if available, std::numeric_limits<Capacity>::max() otherwise. If infty is set to a non-zero value, that value will be used as "infinite".

If the file type was previously evaluated by dimacsType(), then the descriptor struct should be given by the dest parameter.

This function reads a maximum flow instance from DIMACS format, i.e. from a DIMACS file having a line starting with

p max

At the beginning, g is cleared by g.clear(). The arc capacities are written to the capacity arc map and s and t are set to the source and the target nodes.

If the capacity of an arc is negative, it will be set to "infinite" instead. The actual value of "infinite" is contolled by the infty parameter. If it is 0 (the default value), std::numeric_limits<Capacity>::infinity() will be used if available, std::numeric_limits<Capacity>::max() otherwise. If infty is set to a non-zero value, that value will be used as "infinite".

If the file type was previously evaluated by dimacsType(), then the descriptor struct should be given by the dest parameter.

This function reads an arc capacitated digraph instance from DIMACS 'max' or 'sp' format. At the beginning, g is cleared by g.clear() and the arc capacities/lengths are written to the capacity arc map.

In case of the 'max' format, if the capacity of an arc is negative, it will be set to "infinite" instead. The actual value of "infinite" is contolled by the infty parameter. If it is 0 (the default value), std::numeric_limits<Capacity>::infinity() will be used if available, std::numeric_limits<Capacity>::max() otherwise. If infty is set to a non-zero value, that value will be used as "infinite".

If the file type was previously evaluated by dimacsType(), then the descriptor struct should be given by the dest parameter.
