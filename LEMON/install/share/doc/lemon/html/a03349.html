LpBase::ColIt Class Reference

Detailed Description

Its usage is quite simple, for example, you can count the number of columns in an LP lp:

int count=0;
for (LpBase::ColIt c(lp); c!=INVALID; ++c) ++count;

#include <lemon/lp_base.h>
