Detailed Description

template<typename M1, typename M2>
class lemon::LessMap< M1, M2 >

This read-only map assigns true to the keys for which the corresponding value of the first map is less then the value of the second map. Its Key type is inherited from M1 and its Value type is bool. M2::Key must be convertible to M1::Key.

If m1 is of type M1 and m2 is of M2, then for

LessMap<M1,M2> lm(m1,m2);

lm[x] will be equal to m1[x]<m2[x].

The simplest way of using this map is through the lessMap() function.
